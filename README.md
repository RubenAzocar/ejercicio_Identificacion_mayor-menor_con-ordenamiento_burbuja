# Ordenamiento Burbuja - Identificación del Mayor y Menor

## Descripción

Programa en JavaScript que solicita 3 números al usuario, los ordena usando el algoritmo de ordenamiento burbuja con ciclo do-while, y determina el número mayor y menor.

## Estructura del Proyecto

```
ejercicio_Identificacion_mayor-menor_con ordenamiento_burbuja/
├── index.html
├── README.md
├── assets/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── app.js
```

## Cómo Usar

1. Abrir `index.html` en el navegador
2. Ingresar 3 números cuando se solicite
3. Ver los resultados en pantalla

## Características

- Solicitud de 3 números con `prompt()`
- Conversión con `parseFloat()`
- Almacenamiento con `array.push()`
- Validación con `isNaN()`
- Ordenamiento burbuja con `do-while` y `for`
- Identificación del mayor y menor
- Manejo de números iguales
- Visualización con `innerHTML` y `alert()`
